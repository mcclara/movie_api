MyMovieConnect!

This app will allow users the opportunity to create profiles where they are able to compile a list of their favorite films. They will also be able to search films and access the films metadata.

Business Logic Endpoint (URL) HTTP Method Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None A Json Object with data about all of the movies
Return data about a movie by it's title /movies/:title GET None A Json object with data about a single movie's description, genre, director, img URL and whether it is a feature film
Return a list of films by genre /movies/:genre GET None A Json oject with descriptions of genres
Returns the name of the director by film /movies/:director GET None A Json Object with data about director
Allows New Users to register /users POST A JSON Object that will hold new users data including, name, username, password, date of birth, email address and location A JSON Object with data about the user that was added, including an id
Allows the user to update their username /users/:username PUT A JSON object with users updated username A JSON Object holding users updated data
Allow Users to add a film to their favorite's list /users/:username/movies/:title POST A JSON object with data about the added film A message stating that the film was removed
Allow Users to remove a film from their favorite's list /users/:username/movies/:title DELETE None A message stating that the film was removed
Allow existing users to unregister /users/:username/ DELETE None A message stating that the account was removed
